UVa 10360 - Rat Attack

Contents

  1. 1. Problem
  2. 2. Solution
  3. 3. Code

Problem

題目網址

找出炸彈可以殺掉最多老鼠的位置和數量。

Solution

記錄每一格可以殺的老鼠數量，也就是將可以炸到該群老鼠的位置，都加上當前老鼠數。

另外一種方法則是把老鼠確切的位置存入它的數量，以移動炸彈大小的方格去找出最大和。(比較慢)
