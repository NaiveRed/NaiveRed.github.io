UVa 624 - CD

Contents

  1. 1. Problem
  2. 2. Solution
  3. 3. Code

Problem

題目網址
中文網址

給你音樂，挑選要放到錄音帶上的，並使錄音帶空白的長度越小越好。(不得重複)

p.s. 可能不只一組解

Solution

用 DP 算出每個錄音帶長度可放的最長時間，length[j - CD[i]] + CD[i] > length[j] ，j 為時間 、CD[i] 為每首音樂的長度。
並額外用一個二維陣列紀錄這個時間點有沒有使用某首音樂，use[i][j] ，i 為第幾首音樂、 j 為時間。
