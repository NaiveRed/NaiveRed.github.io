UVa 990 - Diving for Gold

Contents

  1. 1. Problem
  2. 2. Solution
  3. 3. Code

Problem

題目網址
中文網址

給你金幣的位置和數量，求出在有限時間內最多可取多少金幣和順序？
p.s. 每次拿完一處的金幣皆須浮上來

Solution

背包問題，直接做 DP 即可，並記下每一個位置在每個時間的打撈狀況 get[i][j] 。輸出時記得依照輸入順序。
