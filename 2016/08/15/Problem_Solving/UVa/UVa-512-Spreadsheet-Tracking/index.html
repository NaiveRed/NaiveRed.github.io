UVa 512 - Spreadsheet Tracking

Contents

  1. 1. Problem
  2. 2. Solution
  3. 3. Code

Problem

題目網址

刪除和插入多個時，都是同時完成，要注意其位置。

Solution

把每個指令記錄下來，再詢問時每次都跑一次，看詢問的點最後在哪裡，不需要記整個表格。
例如在第 5 列的點，刪除 1~4 列時會使其變成在第4列。
