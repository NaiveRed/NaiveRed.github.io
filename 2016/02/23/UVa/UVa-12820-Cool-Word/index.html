UVa 12820 - Cool Word

Contents

  1. 1. Problem
  2. 2. Solution

Problem

題目網址
給你字串，算有幾個是Cool words!
Cool words 也就是字串中有至少兩種字母，每種字母的字數都要不一樣。

Solution

用 int alphabet[26] 記下26個字母的個數，再用 bool num[31] 判斷數字是否有重複即可，記得處理字母種類只有一種的狀況。
