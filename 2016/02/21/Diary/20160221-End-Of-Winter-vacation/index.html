以寒假為句點

Contents

  1. 1. 前言
  2. 2. 逆水行舟
  3. 3. 度過寒假
  4. 4. 總結

前言

寒假在今天就正式結束了，先前所規劃的事大約完成了50%左右而已，目標也沒有達成，實在慘不忍睹。反覆思考到底問題出在哪，也仍舊不知所以，索性統整一下從寒假前和寒假後到底做了些甚麼，假反省之名，行抒發之實。

逆水行舟

如果只是要抱怨成績上的不足，單單幾句就可以結束了，想當然不能解決問題。在這學期的成績出來時，著實吃驚不已，名次退了，自認為在這學期花費了很多心力，也就是期望中分母過高的情況下，分子卻沒有增長。這是為什麼呢? 其實很好找出答案，成績攤在眼前，哪些科目不滿意，再想想所作所為就可以了。但我也沒有因此得到釋懷，因為得出結論後，還是覺得分數有點太低，敝帚自珍嘛~可惜也想不出自己在這科憑什麼拿到高分。這裡所指為「遊戲設計」的課程，讓我短短的反省一下，大概有三點:

  1. 書面報告不夠完善，文件不夠完整
  2. 口頭報告，表達不佳 (雖然是組員君，但我也沒自信能夠講好就是了)
  3. 遊戲實作，太晚開始，以至於時間毫不留情把我輾過在後

這麼看來好像可以分數化的東西我都沒做好，原來是罪有應得啊。但說到學了些什麼，還是可以列出不少，也至少對得起自己了。

說起其他科，儘管都拿到A+，卻相當清楚的知道自己是愧對於這個科目名稱的，畢竟考得好成績，只是代表考卷上的剛好會罷了(甚至猜對的)。獲得學分和獲得學問，兩者可以並行，但距離卻天差地遠。當然，拿到不錯的成績自然是令人高興的事。

有位學長說:「如果要從實力和分數擇其一，本身就不是強的人了。」，那至少讓我能擇其一吧，也總比兩者皆空好。本來對於營養學分 不屑一顧 ，到今天覺得 是不是該選了 的自己感到十分失望。沒有辦法說明選擇分數是否會走向好結局，只是突然開始偏向了那裏，是因為它相較之下顯而為之且成果易見嗎? 唉!

比起 逆水行舟，不進則退 ，這次就讓翻船的我繼續在水中 掙扎 一陣子吧!

度過寒假

沒有在寒假做到對得起它的事，對不起了!只稍微碰了一些東西:

  • Hexo
    一個blog framework，藉著它也碰到了其它相關的。
    • Git
      因為我是架在github上，想說趁著機會來學學文字介面操作git，不過真的就只學了皮毛，一些基本操作和觀念。
    • html
      把 程式語言教學誌 裡的東西看了看實在沒很感興趣…
    • css
      同上，且絕大部分還是碰到問題後去google…

真的非常感謝 Morris 讓我纏著問了不少問題!

解題方面預想是 UVa 解到200題，藉此來熟悉些演算法等等，這部分是有確實做到，但實際功效如何恐怕不得而知，畢竟還有題目難度等其他因素，對有在解題的人來說恐怕是片蛋糕，但廢材如我，還是忍不住高興的紀念一下。

其他更加零碎的項目就不列出了，有一天沒一天在做得事想必成效不彰。

意外的是對於戒掉 LOL 並沒有特別的不適感(廢話)，寫程式之餘看看動漫、小說，也充分的提高了SAN值。玩樂面雖然沒去哪特別的，倒是挺心滿意足，去玩了幾次桌遊，跟友人逛逛等等。回過頭來，也就開始收尾了。

總結

看到了很多我認為 好強啊~~ 的人事物，而往往忽略其中的過程，有些寫了心路歷程或是手札日記等，看完後有些許共鳴外，才驚覺 啊~別人也是這樣努力過來的 。不免俗的希望下學期能秉持著持之以恆的精神去做事情，畢竟知道自己容易半途而廢，問題始終是毅力不夠，時間上或許也有諸多分配不當的情況，能改善的話就太好了。

在查關於 Tim Cook 的 wikipedia 時偶然發現的一句話，蠻喜歡的XD。

千萬不要認為過去發生的任何一件事，會對你即將完成的大事造成阻礙。 - Tim Cook

用這句話來跨過那些不滿意與充滿悔恨的事，並以寒假為句點結束這學期。
