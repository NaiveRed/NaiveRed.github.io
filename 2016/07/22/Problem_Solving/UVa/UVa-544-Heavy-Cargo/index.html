UVa 544 - Heavy Cargo

Problem

中文網址

p.s. 路徑中最小的那個才是整段路的最大負載重量

Solution

直接做 SPFA 找最大負載重量。

判斷式 : 

int maxW = weight[u] ? MIN(adjM[u][v], weight[u]) : adjM[u][v];

weight[u] 等於零時表還沒計算過，所以直接等於 adjM[u][v] 即可。
兩段路 (st->u->v) 中可負載的最大為，兩段中較小的。
