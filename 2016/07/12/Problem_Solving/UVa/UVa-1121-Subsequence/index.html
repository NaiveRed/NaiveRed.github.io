UVa 1121 - Subsequence

Contents

  1. 1. Problem
  2. 2. Solution
  3. 3. Code

Problem

題目網址

給一串數字，找出連續最少幾個數字可大於等於 $S$ 。

Solution

記錄目前總和 sum及目前使用第一個的數字 [tail]。只要一加到超過 $S$ ，就持續將 sum 減掉 [tail]，減少數字個數，並一邊記錄個數的最小值。
