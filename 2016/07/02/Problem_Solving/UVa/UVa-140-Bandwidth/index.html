UVa 140 - Bandwidth

Contents

  1. 1. Problem
  2. 2. Solution
  3. 3. Code

Problem

題目網址
中文網址

先給定圖，接著找出一組特定排序，使得圖中所有的邊，最大權重為最小。(只要兩點可連通，就要去判斷它在排序內的間隔。)
注意此權重為 排序內的間隔。

Solution

可以先將用到的字母依序編號，字典序小的先，方便待會做排列。
要窮舉出全部的排列，可以採用 DFS 或是 std::next_permutation 。
接著只需找出排列中最大的邊，再從所有排列之中找一組最小的。

找排列中最大的邊:

for (i = 0; i < n; i++)
    for (int j = 0; j < n; j++)
        if (adjM[map[order[i]]][map[order[j]]] && abs(i - j) > max)
            max = abs(i - j);
