UVa 10369 - Arctic Network

Contents

  1. 1. Problem
  2. 2. Solution
  3. 3. Code

Problem

題目網址

利用衛星和 radio 來通訊，計算最小的 D。D 為兩個相隔最遠的城市的距離。
衛星則可無視距離。

Solution

在生成樹時，減少建的邊以衛星代替，利用 Kruskal 先建小的邊即可。
