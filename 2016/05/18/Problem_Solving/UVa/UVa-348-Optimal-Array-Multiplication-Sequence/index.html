矩陣乘積鏈

Solution

學校剛好在教這部分呢~稍微注意一下 index 每次的邊界處理。
DP 的列式基本上就是判斷乘的順序看怎樣 cost 比較小:
dp[i][j] = min(dp[i][j] , dp[i][k] + dp[k + 1][j] + d[i - 1] * d[k] * d[j])
