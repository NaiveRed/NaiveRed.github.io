前言

看完書的當下通常都會有很多想法或感觸，但通常我在大約一星期後，就只能告訴別人這本書不錯看，卻無法更詳細的介紹下去了。
所以來試著寫寫 讀後感，不過畢竟不是專業書評，內容大概只會是把我看書時，所想起的事串在一起罷了，不要期待會有什麼詳細的分析或講解。
大致上分成前半段討論 書的劇情，後面則是我的 自言自語 了。

另外內容基本上是 有雷 的。#暴雷一時爽(ry

正常的簡介請參考 博客來-人生複本。

注意：建議不在意被劇透的人再繼續往下讀。

書的部分

我所知道的故事

大致上在講述主角於各個 平行時空 中奔走，目的在於要找回原本自己所在的那條。那條有他所愛的人、愛他的人存在的地方。
起因則是另一個平行時空的自己，不滿意所在的世界，而肆意搶走了其他平行時空的自己的位置。
有點饒舌，但總之就是不同時空下的自己(A)搶走了自己(B)的人生。

至於方法則牽涉到了量子力學，作者用了 薛丁格的貓 去解釋，
裡面其實對於平行時空等有諸多有趣的比喻和講解，像是池塘中的錦鯉和長廊中無限的門。可以多少補足讀者(X)我(O)在這方面的缺乏。

轉換時空的過程則是這樣：進入箱子 → 打藥 → 進走廊 → 挑一扇門 → 走出去
這之中有許多影響結果的要素，主要就是開門時所帶有的想法。為了回到本來的時空，主角反覆進行上述步驟。
而這些時空又都在某個時間點產生各自的 分歧，持續遞迴下去，就像一顆超大的樹(資料結構意味)。

箱子大概像這樣子運作。大概啦。

void box(life A, type darkmatter)
{
    // darkmatter do something...
    int n = random();
    for(int i = 0; i < n; i++)
        life_tree.add(new life());

    A.goto(life_tree.getLife(A.getNum()));
}

在眾多時空遊走，難免會情緒崩潰，主角剛好配有一名 心理治療師。

(夜晚)
治療師：(只剩內衣褲
主角：(我不能、我不能、我不能…

要在近乎無垠的走廊中找尋原來的時空，就像是在沙灘中尋找一粒沙一樣。關鍵在於開門時所注入的情感和情緒。

治療師：你只寫出了你所記得的世界，卻沒有說出自己的感受。
主角：！！！

然而順利回去後卻發現比被自己 NTR 還嚴重的事。

感想

剛開始閱讀時，馬上就想到了《命運石之門》，雖然在實行方式和具體設定上不同，但終究跟妹子(嘟嘟嚕)有莫大關係。(順便推薦一下該部動畫
至於哪個主角比較慘，我想是倫太郎略勝一籌。
等等等，還有《蝴蝶效應》、《魔法少女小圓》應該也都算在拯救妹子。(雖然我沒看完整部

穿越時空、平行世界的題材絕對不嫌少，也因此本書某些橋段顯得有些老梗，或是說容易預見的情節。
但並不影響整體閱讀的樂趣，實際上很容易一口氣讀完，十分引人入勝。該高潮、該失落的地方都還算是容易帶入情感。

其中最扣我心弦的段落是主角在各個時空穿梭時的種種困境，究竟在自己面前的愛人是不是自己所愛的那個人呢？
同個時空的數個自己，到底哪個是真的？
突然想起先前《哲學哲學雞蛋糕》中看到的 雷修斯的船 。
雖然可能有點不同，但這本書確實有不少哲學色彩。

超級雷警告：
另外一段高潮應該在回到自己的時空時所看到的無數個自己，每個人都想搶回自己的生活，究竟該怎麼解決？
解法有點小失望，至少我認為其他的自己並不會這樣善罷甘休。

你要承擔自己的選擇，從中學得教訓，而不是投機取巧。 - 《人生複本》

這應該是書中想傳達的其中一件事情，認真地活在當下，別肖想重新開始，除非你物理很強。

之所以有現在的自己，便是透過人生中一連串的 選擇 所形成的。至於結果好壞只有自己能判斷，不斷懷疑，我想也是選擇的一部分。

另外令人燒腦頭痛的不是劇情本身，而是裡面用到的理論才對，不深入探討也能順利閱讀。#認真就輸了
所以不要被書腰騙了，本來看到標著燒腦神作，心中想著好麻煩啊，不想動腦…

推薦給曾想過「如果當初我…」的人。
實際上你或許已經做了只是在別的時空罷了。

自言自語

如果要談起看書時的所有悸動和所聯想到的事，就太花費篇幅了。
只是想提一下關於國中時的 小小戀歌，非常片段，畢竟也不是本篇的主題。

之所以說是小小戀歌，是因為時間上兩人交往的時間的確很短，所經歷到的事也不算太多。
但正因如此，每件事都歷歷在目、無法忘懷，並留下了不少遺憾和無奈。啊！就此打住。

(擷取書中的內容)

那正是青春的美好之處，一切都瀰漫著一種失重感，還沒有做出毀滅性的選擇，還沒有選定道路，前方岔路純粹只代表了無限可能性。

要說選定道路，到目前為止絕對做了數不清的決定，並走上了某一條通往現在的路，像是選定了高中、大學、讀的科系等等。但似乎還沒出現關鍵性的一筆。
即使到了現在要決定研究所，前方仍有無限可能。

但回顧兩人交往的過程中，的確有毀滅性的選擇，也就是分手。
對那時的我來說，就算事後試圖挽回，前方仍沒有任何岔路，全都筆直的通往了阿比斯深淵。

我想她偶爾午夜夢迴，也會懷疑自己當初選擇的路對不對，會擔心自己選錯路，會好奇和我在一起的人生會是什麼樣子。

很好奇她會不會，至少我會。

我想過無數次 如果 當時沒提分手，我們之後會如何呢？能如約定般一起讀高中嗎？高中之後呢？
還是也會因為其他事情導致各自分飛，在畢業前走向一樣的結果。

我能期待別的世界線有我和你還在一起的故事嗎？
就算有，我也得重修物理和專攻量子力學，才能略知一二吧。可惜現在讀的是資工，而且還算喜歡貓。

「這件事該怎麼解決」
「我正在解決」
「什麼意思」
「意思就是明天早上再問我一次」

沒什麼意思，我只是單純很中意這段對話。

後記

Ep，這只是我一時興起，想試著學學大Youtuber時代的說書系列。但想不到要叫什麼，只好先打上 Ep。
不過光是要重新架構書的內容甚麼的，再佐以自己的想法和評論，拍成影片後還要各種後製。說書甚麼的實在有夠難。
